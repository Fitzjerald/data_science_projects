Identify Periods of High Volatility

Next we apply statistical techniques in order to identify periods of high volatility in the exchange rates. This analysis can help uncover specific time periods when the exchange rates experienced significant fluctuations.

# Calculate the volatility (rolling standard deviation) for each currency pair
window_size = 30  # Adjust the window size
data['volatility'] = data.groupby('currency')['value'].rolling(window_size).std().droplevel(0)

# Select a subset of currencies based on volatility
top_currencies = data.groupby('currency')['volatility'].mean().nlargest(10).index

# Filter the data for the selected currencies
subset_data = data[data['currency'].isin(top_currencies)]

Forecasting

We further consider applying time series forecasting techniques to predict future volatility based on historical patterns. This involves autoregressive integrated moving average models (ARIMA).

In [36]:
# Prepare data for forecasting
forecast_data = data[data['currency'].isin(top_currencies)][['date', 'currency', 'volatility']]
forecast_data.set_index('date', inplace=True)

# Perform forecasting and compare with actual values
for currency in top_currencies:
    # Extract the historical volatility for the current currency
    currency_data = forecast_data[forecast_data['currency'] == currency]['volatility']

    # Train-test split
    train_data = currency_data[:-30]  # Use all but the last 10 observations for training
    test_data = currency_data[-30:]  # Use the last 10 observations for testing

    # ARIMA model
    model = ARIMA(train_data, order=(1, 0, 1))  # Adjust the order as per your requirement
    model_fit = model.fit()

    # Forecast future volatility
    forecast = model_fit.forecast(steps=30)  # Forecast the next 10 periods

    # Compare with actual values
    actual_values = test_data.values
    forecast_values = forecast.values

    # Calculate evaluation metrics
    mse = mean_squared_error(actual_values, forecast_values)
    rmse = np.sqrt(mse)

    print("Currency:", currency)
    print("Forecasted Volatility:", forecast_values)
    print("Actual Volatility:", actual_values)
    print("RMSE:", rmse)
    print("--------------------------")
